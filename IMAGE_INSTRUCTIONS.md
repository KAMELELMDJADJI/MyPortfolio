# Image Instructions for Portfolio

## Where to Place Images

All images should be placed in the `images/` folder in your project root.

## Required Images

### 1. Home Section Image
- **Location**: Already exists at `images/home.jpg`
- **Current**: Your home image is already set up
- **Size Recommendation**: 600x600px or larger, square or portrait orientation

### 2. About Section Image
- **Location**: Already exists at `images/about.jpg`
- **Current**: Your about image is already set up
- **Size Recommendation**: 500x500px or larger, square orientation

### 3. Skills Section Image (Optional)
- **Location**: `images/skills.jpg`
- **Current**: Not required, but you can add one if you want
- **Size Recommendation**: 800x400px, landscape orientation

### 4. Project Images (NEW - Required for Projects Section)
You need to add 3 project images:

- **Project 1**: `images/project1.jpg`
  - This will be displayed in the first project card
  - Size: 400x300px or larger, landscape orientation
  - Format: JPG, PNG, or WebP

- **Project 2**: `images/project2.jpg`
  - This will be displayed in the second project card
  - Size: 400x300px or larger, landscape orientation
  - Format: JPG, PNG, or WebP

- **Project 3**: `images/project3.jpg`
  - This will be displayed in the third project card
  - Size: 400x300px or larger, landscape orientation
  - Format: JPG, PNG, or WebP

## Image Naming Convention

- Use lowercase letters
- Use underscores or hyphens for spaces (e.g., `project_1.jpg` or `project-1.jpg`)
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

## Notes

- If an image is missing, a placeholder icon will be displayed automatically
- Images will automatically resize to fit the layout
- For best results, use high-quality images (at least 72 DPI for web)
- Optimize images before adding them to reduce page load time

## How to Update Project Images

1. Add your project images to the `images/` folder
2. Name them exactly as: `project1.jpg`, `project2.jpg`, `project3.jpg`
3. The images will automatically appear in the Projects section
4. If an image doesn't load, a placeholder will show instead
